# LectureLight Web

A university study planner for tracking courses, lectures, progress, and dark mode preferences.

## Run locally

```bash
npm install
npm run dev
```

Open the local URL shown in the terminal, usually `http://localhost:5173`.

## Build

```bash
npm run build
npm run preview
```

## Deploy to Vercel

This project includes `vercel.json` so Vercel can install dependencies and run the build automatically.

```bash
npx vercel
```

For production:

```bash
npx vercel --prod
```
